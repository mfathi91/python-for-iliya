# تمرین ۱
یک برنامه بنویسید که محتویات فایل زیر را بخواند و در نهایت چاپ کند که کدام عدد بزرگترین عدد موجود در فایل است (هر خط فقط حاوی یک عدد است):
```
3000
5000
1500
4200
4800
900
80
3500
4150
2000
```
خروجی:
```
5000
```

# تمرین ۲
یک برنامه بنویسید که محتویات فایل زیر را بخواند و در نهایت چاپ کند که هر رشته دارای چند حرف m است (هر خط فقط حاوی یک رشته است):
```
mom
dad
mina
momo
mummy
folder
sina
mohsen
```
خروجی:
```
mom: 2
dad: 0
mina: 1
momo: 2
mummy: 3
folder: 0
sina: 0
mohsen: 1
```
